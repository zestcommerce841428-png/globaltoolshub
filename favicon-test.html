Favicon Information

The site uses favicon.svg and logo.svg for icons.

For best compatibility, add these tags to all HTML pages:

<link rel="icon" type="image/svg+xml" href="/globaltoolshub/favicon.svg">
<link rel="icon" type="image/svg+xml" href="/globaltoolshub/assets/logo.svg">
<link rel="apple-touch-icon" href="/globaltoolshub/assets/logo.svg">
