FAQ

Can I refresh any webpage? ▼

Due to browser security restrictions, some websites don't allow being embedded in iframes. You can use background request mode or new window mode to refresh such websites.

What is the minimum refresh interval? ▼

The minimum is 1 second, but we recommend setting it to 5 seconds or more to avoid putting pressure on the target website.

Will refreshing continue after closing the page? ▼

No, the refresh function requires the page to remain open. After closing the browser tab, the auto-refresh will stop automatically.

Can I refresh multiple pages simultaneously? ▼

Yes, you can open multiple tool tabs and set different URLs to refresh simultaneously.

What are the use cases for this tool? ▼

It's suitable for monitoring page updates, ticket grabbing, flash sales, and monitoring data changes.

Is this tool really free? Is it safe? ▼

Yes, this tool is 100% free with no ads, no signup, and no hidden charges. It runs entirely in your browser.
