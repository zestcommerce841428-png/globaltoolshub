FAQ

            

                

                    
What is WHOIS?

                    
WHOIS is a protocol for querying domain registration information, allowing you to view domain owner, registrar, and other details. Through WHOIS queries, you can learn about domain registration time, expiration time, DNS server configuration, and registrant contact information.

                

                

                    
What is the difference between WHOIS and RDAP?

                    
RDAP (Registration Data Access Protocol) is the modern replacement for the traditional WHOIS protocol. RDAP returns structured JSON data, supports HTTPS encrypted transmission, provides internationalized content, and follows standardized access control. Our tool uses RDAP for more reliable and comprehensive query results.

                

                

                    
Which domain extensions are supported?

                    
This tool supports most popular domain extensions including .com, .net, .org, .cn, .io, .dev, .app, and many country-code TLDs. The query is performed via RDAP protocol, which covers the vast majority of domain registries worldwide.

                

                

                    
Why is some information hidden or redacted?

                    
Many domain registrars now offer WHOIS privacy protection services that redact personal information like owner name, email, and phone number. This is done to comply with privacy regulations like GDPR. You may see "REDACTED FOR PRIVACY" or similar notices in the query results.

                

                

                    
How to check when a domain expires?

                    
Simply enter the domain name in the search box and click "Query". The result will show the registration date, expiration date, and remaining days until expiration. You can use this information to monitor domain renewals or check domain availability.

                

                

                    
Is this tool really free? Is it safe?

                    
Yes, this tool is 100% free with no ads, no signup, no usage limits, and no watermarks. All queries are made directly from your browser to the RDAP servers — your data never passes through our servers. It's completely safe and private.
